## 🗂️ How the Kanban Board Works

The board gives you a clear view of all work items. Each card usually stands for one issue or task.

Common columns include:
- Backlog
- Ready
- In Progress
- Review
- Done

A simple workflow looks like this:
1. Add a task card.
2. Assign it to an agent.
3. Move it to In Progress.
4. Check progress while the agent works.
5. Move it to Review when the task is ready.
6. Mark it Done after the result looks correct.

This layout helps you see what is waiting, what is active, and what is finished.

## 🔄 Daily Workflow

A simple daily routine can look like this:

1. Open harness-kanban in your browser.
2. Review new issues.
3. Move ready work into the active column.
4. Assign each task to an agent.
5. Check progress through the day.
6. Move finished work to Done.
7. Revisit blocked cards and add notes.

This keeps the board current and helps you spot delays fast.

## 🧹 Tips for Better Organization

Use these habits to keep the board useful:
- keep one task per card
- avoid long card titles
- add short notes for each step
- group related tasks in one board
- archive old cards when they are done
- review blocked work each day

A clean board is easier to use and easier to trust.
